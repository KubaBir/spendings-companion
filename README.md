# Spendings Companion

A simple dashboard for managing your finances. Integrates with the Nordigen API to sync your banking account data and transations.

## Stack:
- Next.js
- Clerk
- Shadcn
- Tailwind CSS
- Mongo DB

Nordigen (currently [GoCardless](https://gocardless.com/g/gc-nordigen/)) handles integrating with multiple bank APIs. \
This app supports authorizing with Clerk and connecting your bank account via a guided process.

## Getting Started

```bash
pnpm i
pnpm dev
```
